Grid

The Compose grid uses flexbox to create columns. You can initiate the grid by adding a class of cdk-grid to a containing element (usually a <div>). This will cause the elements inside the container to be lined next to each other. Elements inside the grid, will have the size of their, unless you specify a column width.

Compose allows for a grid of up to 6 columns, and sets a column gutter of 2% of the container size between each column. To set a column width use a class starting with cdk-col- and ending with a number between 1 to 6. The number will indicate how many columns you want your element to span. For example, an element with a class of cdk-col-3 will span 3 out of 6 columns (or half of its container). If you do not set a number to the cdk-col class, then it will span 6 columns (or the full width of the container). 

<div class="cdk-grid">
    <div class="cdk-col-3 cdk-card">
        <p>3 Cols</p>
    </div>

    <div class="cdk-col-3 cdk-card">
        <p>3 Cols</p>
    </div>
</div>

Responsive Column Widths

You may want the width of your columns to vary between devices. In order to do this, you will need to add the suffixes -md and -lg to your column classes. These two suffixes correspond to the Compose medium and large responsive breakpoints respectively.

Columns With Multiple Width Breakpoints

Using a class of cdk-col-3 will give your element a width of 3 columns for all devices. However, you may want the element to span 2 columns for tablets and 1 column for desktops. If this is the case, you can use the class suffixes -md and -lg to set breakpoints for your columns. If we follow this example, you could have an element with three width classes: class="cdk-col-3 cdk-col-2-md cdk-col-1-lg" 

<div class="cdk-grid">
    <div class="cdk-col-3 cdk-col-2-md cdk-col-1-lg cdk-card">
        <p>3 Cols Sm, 2 Cols Md, 1 Col Lg</p>
    </div>

    </div>

Columns With One Width Breakpoint

It may be possible that you want the size of your column to only change once. For example, you may want it to span 6 columns in mobile and 2 columns in medium and large devices. If that is the case, you only need to set the highest breakpoint. By default, any class starting with cdk-col spans 6 columns up until its breakpoint is reached. This means that an item with class="cdk-col-2-md will be 6 columns in small devices, and 2 columns in medium and large devices. It also means that an item with class="cdk-col-3-lg will be 6 columns in small and medium devices and 3 columns in large devices. 

<div class="cdk-grid">
    <div class="cdk-col-2-md cdk-card">
        <p>6 Cols Sm, 2 Cols Md and Lg</p>
    </div>

    <div class="cdk-col-2-md cdk-card">
        <p>6 Cols Sm, 2 Cols Md and Lg</p>
    </div>

    <div class="cdk-col-2-md cdk-card">
        <p>6 Cols Sm, 2 Cols Md and Lg</p>
    </div>

    <div class="cdk-col-3-lg cdk-card">
        <p>6 Cols Sm and Md, 3 Cols Lg</p>
    </div>

    <div class="cdk-col-3-lg cdk-card">
        <p>6 Cols Sm and Md, 3 Cols Lg</p>
    </div>
</div>
