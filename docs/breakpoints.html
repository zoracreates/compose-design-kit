Responsive Breakpoints

Compose was developed mobile-first and uses media queries to set breakpoints for its layout and typography. The breakpoints are as follow:

  • Large Devices: ≥56.25em (900px)
  • Medium Devices: ≥48em (768px)
  • Small Devices: <48em

If you would like to change the value of a breakpoint, you may do so by changing the values of the $large-devices and/or $medium-devices Sass variables in the _variables.scss partial file.
