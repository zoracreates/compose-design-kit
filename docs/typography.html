Title and Body Fonts

Compose uses two different font-families: Playfair Display for headings (h1 - h6), and Roboto for all other text. You can change these font-families by changing the Sass variables $title-font and $body-font in the _variables.scss partial file.

Typescale

By default Compose uses a Major Third typescale. This means that font-sizes increment by 1.25. In devices with a screen size of 48em (768px) or less, any body text (non-heading text elements such as paragraphs and lists) have a font-size of 1rem, and heading elements (h1 - h6) are based on an increment of 1.25. In devices larger than 48em, the body text is 1.25rem, and the heading elements increase from that base. The lists below show how this looks like.

Note that the document root font-size for Compose is 16px, and this establishes the value of 1rem (1rem = 16px and 1.25rem = 20px).

Small Screens Scale—48em (768px) or smaller

  • Paragraphs and lists: 1rem (16px)
  • Heading 6: 1rem x 1.25 = 1.25rem (20px)
  • Heading 5: 1.25rem x 1.25 = 1.563rem (25px)
  • Heading 4: 1.563rem x 1.25 = 1.953rem (31px)
  • Heading 3: 1.953rem x 1.25 = 2.441rem (39px)
  • Heading 2: 2.441rem x 1.25 = 3.052rem (48px)
  • Heading 1: 3.052rem x 1.25 = 3.815rem (61px)

Large Screens Scale—larger than 48em (768px)

  • Paragraphs and lists: 1.25rem (20px)
  • Heading 6: 1.25rem x 1.25 = 1.563rem (25px)
  • Heading 5: 1.563rem x 1.25 = 1.953rem (31px)
  • Heading 4: 1.953rem x 1.25 = 2.441rem(39px)
  • Heading 3: 2.441rem x 1.25 = 3.052rem (48px)
  • Heading 2: 3.052rem x 1.25 = 3.815rem (61px)
  • Heading 1: 3.052rem x 1.25 = 4.769rem (76px)

Changing the Default Scale

Compose uses CSS custom properties to create a customizable responsive typescale. The font-size for body text elements is set by the --base-font-size CSS custom property. The rest of the font-sizes build off --base-font-size, and they increase at a ratio set by the --multiplier CSS custom property. You can checkout all typescale custom properties in the _typescale.scss partial file.

As suggested by the lists of font-sizes above, the default --multiplier property is set to 1.25, and the default --base-font-size is set to 1rem for devices with a screen size of 48em (768px) or smaller and 1.25rem for devices larger than 48em.

The easiest way to change the typescale is by changing the value of --base-font-size and/or --multiplier at the CSS :root pseudo-class 

//change the font-scale for devices 500px or bigger
@media (min-width: 500px) {
    :root {
        --base-font-size: .75rem; //12px
        --multiplier: 1.414;
    }
}   

//results
    // body, p, ul, ol: .75rem (12px)
    // h6: .75rem x 1.414 =  1.414rem (16px)
    // h5: 1.414rem x 1.414 = 1.999rem (23px)
    // h4: 1.999rem  x 1.414 = 2.827rem (33px)
    // h3: 2.827rem x 1.414= 3.998rem (47px)
    // h2: 3.998rem x 1.414 = 5.653rem (67px)
    // h1: 5.653rem x 1.414 = 7.993rem(95px)
