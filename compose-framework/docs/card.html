Card

By default a card has a white background and a slight drop shadow. It also has a slight padding to keep elements away from the edges. To use this component add a class of cdk-card. One common way to use cards is by making them into columns of the Grid component. 

<div class="cdk-grid">
    <div class="cdk-card cdk-col-2-md">
        <h3>Card Preview</h3>
        <p>Lorem ipsum dolor sit amet</p>
        <button class="cdk-btn-primary">Button</button>
    </div>

    <div class="cdk-card cdk-col-2-md">
        <h3>Card Preview</h3>
        <p>Lorem ipsum dolor sit amet</p>
        <button class="cdk-btn-primary">Button</button>
    </div>

    <div class="cdk-card cdk-col-2-md">

    </div>
</div>
