Color Theme Classes

You can change the color scheme of different elements in your website by choosing from several Compose color themes. To apply a color theme to an element add one of the following classes: cdk-theme-default, cdk-theme-light, cdk-theme-dark, or cdk-theme-gradient.

If you do not use any of the color theme classes, the color scheme will be the same as cdk-theme-default.

Changing Global Colors

Compose uses Sass variables to define colors. One way to easily customize Compose is by changing the value of its color variables. You can do this directly in the Compose _variables.scss partial file.

Considering Derived Colors

Some Compose colors are based on other colors. For example, changing the value of the $primary-color variable, will automatically change the value of $primary-light, $primary-light-analogous, and $primary-dark. This change happens because Compose uses Sass color functions to calculate the values $primary-light, $primary-light-analogous, and $primary-dark.

The code below comes from _variables.scss and shows how the values are calculated. Note that $primary-light and $primary-dark directly depend on $primary-color. However, $primary-light-analogous is based on $primary-light. Also notice the $color-variation variable, which is used to determine how much a color will be darkened or lightened. 

//COLOR SHADE CALCULATION VARIABLE
$color-variation: 7%;

//PRIMARY COLOR SHADES AND DERIVATIVES
//set $primary-color
$primary-color: #0170A7;

//darken $primary-color
$primary-dark: darken($primary-color, $color-variation); 

//lighten $primary-color 
$primary-light: lighten($primary-color, $color-variation); 

//adjust the hue of $primary-light to an analogus color
$primary-light-analogous: adjust-hue($primary-light, -15);

Color Relationships

This table shows which colors are used to determine other colors within Compose.
Base Color Derived Colors

  • $primary-dark
  • $primary-light
    • $primary-light-analogous
$secondary-color
  • $secondary-light
  • $secondary-dark
$dark-gray
  • $black
$danger-color
  • $danger-dark
