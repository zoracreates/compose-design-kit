Code Block

To use a styled code block like the ones across this documentation website, wrap your code in <pre><code> brackets. Remember to use escape characters for brakets ( &lt; for < and &gt; for >) 

<pre><code>
    &lt;p&gt;Sample Text&lt;/p&gt;
</code></pre>
<pre><code>
    <p>Sample Text</p>
</code></pre>

Code Quote

To use a styled one line of code block like the ones across this documentation website, add a class of cdk-code-quote to a <span> or <code> element. Remember to use escape characters for brakets ( &lt; for < and &gt; for >) 

<code class="cdk-code-quote"> &lt;p&gt; </code>
<p>
