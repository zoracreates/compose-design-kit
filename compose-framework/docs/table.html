Table

Compose provides background-color, border, and font styles for tables. To use a Compose table, add a class of cdk-table to a <table> element.

Note that all <th> elements are bolded by default. Furthemore, any <th> elements with scope="col" will be distinguished by a blue background and white text. 

<table class="cdk-table">
    <caption>
        This is a styled table caption for a table preview.
    </caption>

    <thead>
        <tr>
            <th scope="col">Col Heading</th>
            <th scope="col">Col Heading</th>
            <th scope="col">Col Heading</th>
        </tr>
    </thead>

    <tbody>
        <tr>
            <th scope="row">Row Heading</th>
            <td>Data</td>
            <td>More data</td>
        </tr>

        <tr>
            <th scope="row">Row Heading</th>
            <td>Data</td>
            <td>More data</td>
        </tr>

        <tr>
            <th scope="row">Row Heading</th>
            <td>Data</td>
            <td>More data</td>
        </tr>
     </tbody>
</table>
