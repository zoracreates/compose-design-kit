Accessible More Links

Users of screen readers often have the links on a page read out loud to them. This usually allows them to scan websites more quickly, unless the site has too many non-descriptive links, such as multiple “Read More” links. However, “Read More” links can help some sighted users to scan pages for clickable items. While there are other ways to make items look clickable, sometimes the “Read More” is difficult to avoid. For those difficult situations, Compose created an accessible “Read More” link inspired by Matt Wilcox’s work.

These links allow you to include descriptive text for screen readers, and show “Read More” for sighted users. To use these links simply add the classes cdk-more-link and cdk-default-more-text to an <a>, and include the descriptive screen reader text within a <span> with a class of cdk-sr-only inside the <a>. You can also add an arrow to the end of the link by using an <i> with a class of cdk-arrow and the aria-hidden attribute. 

<a class="cdk-more-link cdk-default-more-text">
    <span class="cdk-sr-only">
        Empty link screen reader only text example
    </span>
    <i class="cdk-arrow" aria-hidden></i>
</a>
