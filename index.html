Install Compose

To get started with Compose, download the files, unzip them, and copy compose.css into your CSS folder. Next, include the following lines in the <head> of your HTML 

<!--Compose Design Kit Fonts-->
<link href="https://fonts.googleapis.com/css?family=Montserrat" rel="stylesheet"/>
<link href="https://fonts.googleapis.com/css?family=Playfair+Display" rel="stylesheet"/>

<!--Compose Design Kit CSS-->
<link rel="stylesheet" href="css/compose.css"/>
